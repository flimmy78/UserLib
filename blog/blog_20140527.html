Smart Pointers

In the standard library of C++11 you find the following smart pointers: unique_ptr, shared_ptr and weak_ptr. unique_ptr is an advancement of old auto_ptr. auto_ptr has a copy semantic, while unique_ptr has a modern move semantic. shared_ptr and weak_ptr implement the counted reference pointer idiom.

UserLib contains class CCppObject, which is the base class of all classes allocated on the heap.

Containers

The standard template library supports four categories: containers, algorithms, functors and iterators. Containers can be divided into two groups: sequential and associative containers. Sequence containers include vector, deque, and list. Associative containers include set, multiset, map, and multimap.

UserLib contains several templated wrapper classes. Wrapper classes of a interface and implementation of data structures written in C. This list include CDataVector, CDataDoubleLinkedList, CDataBinaryTree and CDataBTree. Each class defines its own iterator. And the set of methods forming the interface are very similar to each other. For example the interface of CDataVector is as follows: bool Open(TListCnt cnt, TListCnt exp); TListCnt Count() const; void Close(TDeleteFunc freeFunc = NULL, Pointer context = NULL); Iterator Index(TListIndex index) const { Iterator it = VectorIndex(_liste, index); return it; } Iterator Begin() const; Iterator Next(Iterator node) const; Iterator Prev(Iterator node) const; Iterator Last() const; bool ForEach(TForEachFunc func, Pointer context = NULL) const; Iterator Find(ConstPointer data, TSearchAndSortFunc findFunc) const; Iterator FindUser(ConstPointer data, TSearchAndSortUserFunc findFunc, ConstPointer context) const; Iterator FindSorted(ConstPointer data, TSearchAndSortFunc findFunc = NULL) const; Iterator UpperBound(ConstPointer data, TSearchAndSortFunc findFunc) const; Iterator LowerBound(ConstPointer data, TSearchAndSortFunc findFunc) const; void Sort(TSearchAndSortFunc sortFunc = NULL, TSortMode mode = HeapSortMode) const; void SortUser(TSearchAndSortUserFunc sortFunc, ConstPointer context, TSortMode mode = HeapSortMode) const; Iterator Append(ConstPointer data) const; Iterator Prepend(ConstPointer data) const; Iterator InsertBefore(Iterator node, ConstPointer data) const; Iterator InsertAfter(Iterator node, ConstPointer data) const; void Remove(Iterator node, TDeleteFunc freeFunc = NULL, Pointer context = NULL) const; Iterator InsertSorted(ConstPointer data, TSearchAndSortFunc findFunc = NULL) const; bool RemoveSorted(ConstPointer data, TSearchAndSortFunc findFunc = NULL, TDeleteFunc freeFunc = NULL, Pointer context = NULL) const; ConstPointer GetData(Iterator node) const; void SetData(Iterator node, ConstPointer data) const;
